# Getting Started

Perbug helps facilitators run structured planning sessions in minutes.

## 1. Configure your workspace

Create a workspace, invite participants, and assign facilitator roles.

## 2. Launch your first session

Choose a template, define objectives, and set voting rules.

## 3. Review telemetry

Use basic telemetry to understand participation trends and completion rates.

<Callout type="warning" title="Privacy by default">
Telemetry is optional and should remain disabled unless your organization explicitly enables it.
</Callout>
