# FAQ

## Does Perbug require analytics?

No. Analytics are disabled by default and controlled via environment variables.

## Can I self-host documentation?

Yes. The docs site is static and can be deployed to Vercel, Netlify, Cloudflare Pages, or any static host.

## Where can I report issues?

Use the support email listed on the Support page.
