# Telemetry API

Telemetry endpoints are intended for aggregate product quality insights.

## Ingest event

```http
POST /api/v1/telemetry/events
```

## Disable telemetry

Set `PERBUG_ANALYTICS_ENABLED=false` to keep collection off by default.
