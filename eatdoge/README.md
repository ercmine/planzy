# EatDoge

Standalone meme website for **eatdoge.com** built with React, Vite, TypeScript, Tailwind CSS, and Framer Motion.

## Run locally

```bash
cd eatdoge
npm install
npm run dev
```

## Build

```bash
cd eatdoge
npm run build
```

## Notes

- The app imports the repository's top-level `doge.png` directly; the image is not moved, renamed, or copied.
- The site is intentionally entertainment-only and includes no wallet, trading, staking, or token utility functionality.
