# iOS Permission String Placeholders

If `ios/Runner/Info.plist` is unavailable in this repository snapshot, add these keys before release:

- `NSLocationWhenInUseUsageDescription`
  - Suggested: `We use your location to show nearby plan options for your active session.`
- `NSContactsUsageDescription`
  - Suggested: `We use contacts only to help you invite friends into your plan sessions.`

Keep wording specific, user-benefiting, and aligned with the in-app behavior and App Store policy.
